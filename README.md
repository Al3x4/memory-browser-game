# Front-End Web Developer Nanodegree Program
## Project 3 - Memory Game

This is a 1-page responsive portfolio site with HTML and CSS.

### Description 
 - The game randomly shuffles the cards. A user wins once all cards have successfully been matched
 - 3 levels of difficulty
 - game timer: when the player starts a game, a displayed timer should also start. Once the player wins the game, the timer stops.
 - breaking points for medium (becomes 2 columns) and narrow views (stacked)
 - The game displays a star rating (from 1-3) that reflects the player's performance. At the beginning of a game, it should display 3 stars. After some number of moves, it should change to a 2 star rating. After a few more moves, it should change to a 1 star rating.
 - When a user wins the game, a modal appears to congratulate the player and ask if they want to play again. It should also tell the user how much time it took to win the game, and what the star rating was.

### Features
The number of card on screen chages with difficulty
